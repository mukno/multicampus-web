1. HTML : 웹페이지를 제작하기 위한 도구, 통신규약상의 의미를 말한다
        2. CSS  : HTML 표현한다
		3. 자바 스크립트: 
		         1) HTML 통제하고 유용하게 활용한다 
		         2) 동적화(DOM, Document of Model) 시킬 수 있는 용도 사용한다
		
		4. 자바스크립트 구성 
		   1) 자바스크립트 CORE : 기본 문법과 구조, 데이터 타입, 조건문, 반복문, 함수, 기본 라이브러리 등이 포함 
		   2) 자바스크립트 BOM(Browser Object Model): 브라우저와 관련된 Window, Navigator, Location, History, Document, Screen 객체 
		   3) 자바스크립트 DOM(Document Object Model): Element 제어, Style 제어, Attribute 제어, Event 제어
